# Metadata

The static file `schemas.json` was manually created based on the Aircall API documentation, as Aircall does not provide an OpenAPI spec or a dynamic schema discovery endpoint.

## Supported Objects

The schema includes definitions for the following core objects:

* **Calls** (`/calls`)
* **Users** (`/users`)
* **Contacts** (`/contacts`)
* **Numbers** (`/numbers`)
* **Teams** (`/teams`)
* **Tags** (`/tags`)
