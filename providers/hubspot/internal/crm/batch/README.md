# Implementation not supported

[Create appointments](https://developers.hubspot.com/docs/api-reference/crm-appointments-v3/batch/post-crm-objects-v3-objectType-batch-create) -
requires path parameter `objectType`.

[Create associations](https://developers.hubspot.com/docs/api-reference/crm-associations-v4/batch/post-crm-v4-associations-fromObjectType-toObjectType-batch-create) -
requires path parameters `fromObjectType` and `toObjectType`.

Creating [object properties](https://developers.hubspot.com/docs/api-reference/crm-properties-v3/batch/post-crm-v3-properties-objectType-batch-create) depends on the objectType the properties are created within.
