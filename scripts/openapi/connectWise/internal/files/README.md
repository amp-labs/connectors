# OpenAPI

## Obtain file

The official OpenAPI spec for ConnectWise (e.g., Company API) isn't directly linked but appears in 404 error pages.
1. Navigate to a non-existent product page like: https://developer.connectwise.com/Products/Company
2. On the 404 page, check the `Related search results` section.
3. Look for `Company.txt` or `0.txt` links—these are the OpenAPI files (rename to `openapi.json` after download).

Note: Requires access to the ConnectWise Developer Network (register at https://developer.connectwise.com).
Files are product-specific (Company, Manage, etc.).
